# USDX Logo Design Concepts

## Overview

Five professional logo concepts have been created for the USDX Protocol, each emphasizing different aspects of the project's identity: **hub-and-spoke architecture**, **yield generation**, **cross-chain functionality**, and **stability**.

---

## Design Philosophy

### Core Themes
1. **Hub-and-Spoke Architecture** - Central hub (Ethereum) with radiating spokes (L2 chains)
2. **Yield Generation** - Upward arrows symbolizing growth and returns
3. **Stability** - Solid, trustworthy design elements
4. **Cross-Chain** - Interconnected nodes representing multiple blockchains
5. **The "X"** - Brand identity and mathematical precision

### Color Palette
- **Indigo (#4F46E5, #6366F1)** - Primary brand color, trust, technology
- **Emerald (#10B981, #34D399)** - Growth, yield, prosperity
- **Amber (#FBBF24)** - Energy, value, premium quality
- **Navy (#0A0E27)** - Professional background, depth

## Usage Guidelines

### Do's ‚úÖ
- Use on solid backgrounds (preferably dark)
- Maintain aspect ratio when resizing
- Ensure sufficient clear space around logo
- Use official color palette
- Test legibility at various sizes

### Don'ts ‚ùå
- Don't distort or skew the logo
- Don't change colors arbitrarily
- Don't place on busy backgrounds
- Don't add effects (shadows, glows, etc.)
- Don't use low-resolution versions

### Clear Space
Maintain minimum clear space equal to the height of the "X" in "USDX" around all sides of the logo.

### Minimum Sizes
- **Icon logos:** 32px minimum
- **Wordmark logos:** 150px width minimum
- **Print:** 1 inch minimum width

---

## Color Variations

### Dark Background (Primary)
All provided logos are optimized for dark backgrounds (#0A0E27)

### Light Background
For light backgrounds, invert the color scheme:
- Background: #FFFFFF or light gray
- Primary elements: #0A0E27 (dark navy)
- Accents: Keep indigo, emerald, amber

### Monochrome
For single-color applications:
- **White on dark:** #FFFFFF
- **Dark on light:** #0A0E27 or #4F46E5
